# 🌳order-project
## (주문 프로젝트)
-----------------------------
### 게시판 기능 목록
### 1. 회원 등록 및 조회
### 2. 상품 등록, 조회, 수정
### 3. 주문, 취소, 주문 내역 조회
### 4. 예외처리
- 회원 등록시 중복된 회원이 존재하는가?
- 주문, 취소시 수량이 빠지고 증가하는가?
### 5. 기타 요구사항
- 상품은 재고 관리가 필요하다.
- 상품의 종류는 도서, 음반, 영화가 있다.
- 상품을 카테고리로 구분할 수 있다.
- 상품 주문시 배송 정보를 입력할 수 있다.
### 6. 테스트 케이스 작성
- 기능에 따른 통합 테스트 진행
